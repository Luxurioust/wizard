Classes

AccountController Class for sign-in user account management.
Admin_AnalyticsResource Class for analytics total number of registered users, active users and generate data reports and charts.
Admin_ArticlesResource Class for website CMS system, create, update and destory posts.
Admin_CategoryResource Class for website CMS system, create, update and destory posts category
Admin_ForumResource Class for website forum, include mapping forum category, single post shows, post author create, update and destory the posts and administators management the posts.
Admin_SupportResource Class for administrators management support or suggestions.
Admin_UniversityResource Class for university management, include set opening date, set open, pending and close status.
Admin_UserResource Class for administrators management users, include update, block and destory user, send system notifications to single user or all users.
AdminController Class for web site admin management sytem.
